# High-Density Wi-Fi Architecture & Security Enhancement Lab

## Overview
This project simulates the design of a high-density wireless network architecture for a campus environment, focusing on scalability, performance, and enterprise-grade security controls.

## Environment
- Simulated campus: TriOS College Kitchener Campus
- Target: 1,000+ concurrent users
- High-density wireless deployment

## Tools & Technologies
- Wi-Fi 6 / Wi-Fi 7 (802.11ax / 802.11be)
- VLANs & Network Segmentation
- WPA3 Security
- 802.1X Authentication (RADIUS)
- Microsoft Excel (Planning)
- Visio / draw.io (Architecture Design)

## Key Activities
- Analyzed existing wireless infrastructure and identified scalability and performance gaps  
- Designed high-density Wi-Fi architecture supporting 1,000+ concurrent users  
- Proposed Wi-Fi 6/7 deployment with beamforming and MU-MIMO for improved performance  
- Implemented secure authentication model using WPA3 and 802.1X (RADIUS)  
- Designed VLAN-based segmentation strategy to improve network security and client management  
- Developed cost-effective infrastructure proposal aligned with throughput, VoWiFi, and security requirements  

## Outcome
Designed a scalable and secure wireless architecture that improves performance, enhances user capacity, and strengthens network security using modern Wi-Fi standards and segmentation strategies.

## Project Output
- Wireless Network Architecture Diagram  
- VLAN Segmentation Design  
- Security Configuration Plan (WPA3 / 802.1X)  
- Capacity Planning & Cost Analysis (Excel)
